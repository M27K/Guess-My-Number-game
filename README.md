# Guess My Number! 🎲

A fun and interactive number guessing game built with HTML, CSS, and JavaScript. Test your intuition and see if you can guess the hidden number!

## 🎮 How to Play

1. The game selects a random secret number between 1 and 20.
2. Enter your guess in the input box and click **Check!**.
3. The game will give you feedback:
   - 📈 **Too high!** if your guess is above the secret number.
   - 📉 **Too low!** if your guess is below the secret number.
   - 🎉 **Correct Number!** if you guess it right. The background will turn green!
4. **Score:** You start with 20 points. Every wrong guess deducts 1 point.
5. **Highscore:** The game keeps track of your best score during the session. Try to beat it!
6. Click the **Again!** button at the top left to reset the game and play another round without losing your Highscore.

## 🛠️ Technologies Used

* **HTML5:** For the structure and layout of the game.
* **CSS3:** For styling, colors, and layout positioning.
* **JavaScript (Vanilla):** For DOM manipulation, game logic, and event handling.
